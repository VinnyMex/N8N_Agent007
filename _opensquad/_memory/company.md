# Agent007n8n - Company Profile

## Descrição
Agent007n8n é um sistema de monitoramento e gestão inteligente de fluxos n8n. O objetivo é monitorar execuções de workflows e permitir que um agente de IA resolva problemas, modifique e crie novos fluxos diretamente no n8n através de MCP (Model Context Protocol) e API.

- Criação e modificação de fluxos via MCP e API
